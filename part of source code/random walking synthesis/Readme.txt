Early version of a tool intended to be used for the creation of morphable motion graphs.
Currently only implements some basic functions.
For random walking generation, just run main.py script, then input the step number in pop-up console window.


Installation:

Pre-requires:
Python 2.7

Third party libraries:

matplotlib
NumPy
scikit-learn
cgkit


Precompiled packages can be downloaded from these sites

http://code.google.com/p/pythonxy/wiki/StandardPlugins
